## 🚀 Projetos em Destaque

### [DocScan AI](https://github.com/oFelixJpedro/docscan-ai)
Ferramenta web que detecta documentos em fotos via IA, aplica correção de perspectiva com algoritmo de homografia implementado do zero e exporta em JPG ou PDF.
`JavaScript` `Google Gemini API` `Computer Vision` `jsPDF`

### [Nano Banana Camera Tool](https://github.com/oFelixJpedro/nano-banana-camera-tool)
Ferramenta de reposicionamento de câmera via IA — gera novas perspectivas de qualquer imagem usando modelos de geração de imagem do Google Gemini.
`JavaScript` `Google Gemini Image API` `HTML/CSS`
